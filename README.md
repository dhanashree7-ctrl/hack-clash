# PulseSphere AI

Welcome to the PulseSphere AI monorepo! This repository contains the full-stack architecture for our hackathon project.

## Project Structure

- `/frontend` - Next.js (App Router), TypeScript, Tailwind CSS, and shadcn/ui.
- `/backend` - FastAPI Python backend with Supabase and Transformers integration.
  - `/api` - API routes
  - `/core` - CVI engine logic
  - `/db` - Supabase connections
